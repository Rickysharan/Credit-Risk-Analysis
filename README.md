# Credit Risk Analysis

## Overview

A machine learning project focused on predicting loan default risk using financial and customer data.

## Business Problem

Financial institutions must assess the likelihood that a borrower will repay a loan. Poor risk assessment can lead to financial losses and increased default rates.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Risk Modelling
- Model Evaluation

## Project Status

🚧 Currently under development

## Expected Outcomes

- Predict loan default probability
- Identify key risk factors
- Support credit decision-making
- Improve risk assessment processes

## Career Relevance

This project demonstrates practical applications of data science in banking, lending and financial risk management.
